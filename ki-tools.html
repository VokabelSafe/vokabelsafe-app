🤖 Vokabelsets mit KI-Tools erstellen

Künstliche Intelligenz (wie ChatGPT, Gemini oder Claude) ist der ultimative Abkürzungsweg. Anstatt Vokabeln mühsam abzutippen, lassen Sie die KI direkt eine Datei im VokabelSafe-JSON-Format erstellen.

✨ Der "Direkt-Import"-Trick

Wenn Sie die KI bitten, das Ergebnis direkt als JSON-Code auszugeben, sparen Sie sich den Umweg über Excel oder den Converter. Sie kopieren den Code, speichern ihn als Datei und fertig!

🚀 Beispiel 1: Vokabeln zu einem Thema

Perfekt für den nächsten Urlaub oder ein neues Hobby. Kopiere diesen Prompt:

Erstelle eine Liste mit 20 englischen Vokabeln zum Thema "Im Hotel".
Gib das Ergebnis ausschließlich als JSON-Array aus.
Jedes Objekt im Array muss diese Struktur haben:
{
  "vocabulary_group": "Hotel",
  "term_learning_language": "[Englisches Wort]",
  "term_user_language": "[Deutsche Übersetzung]",
  "learning_status": 1
}
Gib nur den rohen JSON-Code aus.

Klicken Sie auf den Button oben, kopieren Sie den Prompt in Ihre KI (z.B. ChatGPT) und speichern Sie die Antwort als .json Datei.

🎨 Beispiel 2: Visuelle Merkhilfen (Emoji-Power)

Bilder helfen dem Gehirn, Wörter schneller zu speichern. Lass die KI die Emojis direkt in die Übersetzung einbauen:

Erstelle eine JSON-Liste mit 15 Adjektiven zum Thema "Gefühle & Charakter" (z.B. happy, brave, tired).
Wichtig: Die deutsche Übersetzung im Feld "term_user_language" muss immer 1-2 passende Emojis enthalten (z.B. "müde 😴" oder "mutig 🦁").
Gruppe: "Charakter".
Format: [
  {
    "vocabulary_group": "Charakter",
    "term_learning_language": "...",
    "term_user_language": "...",
    "learning_status": 1
  }
]
Gib nur den rohen JSON-Code aus.

Klicken Sie auf den Button oben, kopieren Sie den Prompt in Ihre KI (z.B. ChatGPT) und speichern Sie die Antwort als .json Datei.
In der App sieht das Ergebnis nach dem Import der Datei dann so aus:

🧩 Beispiel 3: Der "Rätsel-Modus"

Mach ein Spiel daraus! Die KI schreibt ein kurzes Rätsel auf Englisch, und du musst die Lösung wissen:

Erstelle 10 englische "Was bin ich?"-Rätsel als JSON-Set.
In "term_learning_language" schreibst du eine kurze Beschreibung (z.B. "I am big, grey and have a trunk").
In "term_user_language" schreibst du nur das Lösungswort auf Deutsch (z.B. "Der Elefant 🐘").
Jedes Objekt muss diese Struktur haben:
{
  "vocabulary_group": "Rätsel-Spaß",
  "term_learning_language": "[Rätsel-Text]",
  "term_user_language": "[Lösung]",
  "learning_status": 1
}
Gib nur den rohen JSON-Code aus.

Klicken Sie auf den Button oben, kopieren Sie den Prompt in Ihre KI (z.B. ChatGPT) und speichern Sie die Antwort als .json Datei.
In der App sieht das Ergebnis nach dem Import der Datei dann so aus:

Was mache ich mit dem JSON-Code?
1. Kopieren Sie den von der KI erzeugten Code-Block (alles zwischen [ und ]).
2. Speichern Sie diesen Text in einer neuen Datei mit der Endung .json (z.B. meine_vokabeln.json).
3. Diese Datei können Sie direkt in der VokabelSafe-App importieren.
