✨ VokabelSet Prompt Generator

Konfigurieren Sie hier Ihr Wunsch-Vokabelset. Der Generator erstellt Ihnen einen optimierten Befehl (Prompt), den Sie einfach kopieren und in eine KI Ihrer Wahl (z.B. ChatGPT) einfügen können.

🤖 Welche KI eignet sich am besten für Vokabel-Sets?
• Gemini 1.5 (Google): Aktuell die beste Empfehlung. Hält sich sehr strikt an die Menge und das Format.
• DeepSeek: Sehr leistungsstark, liefert oft sogar mehr als angefordert. Falls der Code abbricht, einfach "fortsetzen" tippen.
• ChatGPT (OpenAI): Bei größeren Mengen stößt ChatGPT oft an das Limit der Antwortlänge. Wir empfehlen Sets bis max. 300 Vokabeln.
• Tipp: Es ist egal, ob die KI den Code schön untereinander oder alles in eine lange Zeile schreibt – der Converter akzeptiert beides!
⚠️ Wichtiger Hinweis zur Zuverlässigkeit
KIs sind keine mathematischen Maschinen. Es kann vorkommen, dass die KI die Anzahl der Vokabeln falsch zählt oder die Ausgabe bei sehr langen Listen abbricht. Nutzen Sie im Converter den Button "JSON prüfen", um das Ergebnis schnell zu kontrollieren.

💡 Beispiel für das Ergebnis

Die KI gibt den Code oft in verschiedenen Formaten aus. Beides ist absolut in Ordnung:

Variante A: Mehrzeilig (gut lesbar) 
[
  {
    "vocabulary_group": "Unit 1",
    "term_learning_language": "apple",
    "term_user_language": "der Apfel 🍎",
    "learning_status": 1
  }
]
Variante B: Einzeilig/Kompakt (Völlig okay!) 
[{"vocabulary_group":"Unit 1","term_learning_language":"apple","term_user_language":"Apfel","learning_status":1}, ...]

Hinweis: Es ist egal, ob die KI Leerzeichen oder Umbrüche einfügt. Solange die Struktur mit den eckigen Klammern [] und den 4 Feldnamen stimmt, kann der Converter es verarbeiten.
