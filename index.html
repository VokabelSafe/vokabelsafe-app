📚 Lernen, das mitwächst

VokabelSafe ist ein Vokabeltrainer, der sich an den Schulalltag Ihres Kindes anpasst. Statt vorgefertigter Wortlisten tragen Kinder genau die Vokabeln ein, die sie tatsächlich lernen müssen.

Die persönliche Datenbank wächst Jahr für Jahr – von Klasse 5 bis zum Schulabschluss.

🛡️ Für Eltern entwickelt

Die App bietet eine ruhige, sichere Lernumgebung ohne Ablenkung:

  • Keine Werbung
  • Keine Accounts
  • Keine Cloud
  • Keine Tracking-Dienste
  • 100 % offline nutzbar

🧠 Intelligentes Lernen

Lernlevel 1–5 mit Wiederholungssystem
Fokus auf neue und schwierige Wörter
Wählbare Schwierigkeitsgrade
Flexible Abfragerichtung
Übersichtliche Wortliste mit Suche
Multi-Set-System für verschiedene Fächer

📂 Multi-Set-System

Erstellen Sie separate Sets für jedes Schuljahr oder jede Sprache:

  • Englisch Klasse 5
  • Französisch Klasse 6
  • Latein Klasse 7

Sets lassen sich jederzeit erstellen, wechseln und verwalten.

🎯 Motivation ohne Druck

Kinder erhalten für Lernerfolge Münzen, die optional für kleine Minispiele genutzt werden können – ohne Zwang, ohne Wettbewerb.

🔒 Datenschutz & Sicherheit

Alle Daten bleiben ausschließlich auf dem Gerät. Es erfolgt keine Übertragung an Server oder Dritte.

💾 Backup & Import

Die gesamte Datenbank kann gesichert und wiederhergestellt werden. Zusätzlich können Vokabellisten im JSON-Format importiert werden.
