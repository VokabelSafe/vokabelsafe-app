🔧 CSV → JSON Converter & Downloader

Wandeln Sie hier Ihre vorbereitete CSV-Datei um oder fügen Sie direkt JSON-Code (z. B. von einer KI) ein, um ihn als Datei zu speichern. Zur Anleitung

💡 Tipps:
• Excel-Nutzer: Falls Umlaute nach dem Import falsch angezeigt werden, speichern Sie Ihre CSV-Datei in Excel unter "Speichern unter" → Dateityp: CSV UTF-8 (kommagetrennt).
• KI-Nutzer: Sie können den JSON-Code einfach so wie er ist in das Feld unten einfügen. Es spielt keine Rolle, ob er schön formatiert oder in einer langen Zeile steht.
