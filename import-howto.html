Import-Anleitung für VokabelSafe

Sie können Vokabellisten bequem am PC vorbereiten und anschließend als JSON in VokabelSafe importieren.

Die Umwandlung erfolgt direkt im Browser. Ihre Datei wird dabei nicht an einen Server hochgeladen.

📄 Anleitung: Vokabelliste aus Excel / OpenOffice importieren

Tabelle erstellen

Erstelle in Microsoft Excel oder LibreOffice Calc eine Tabelle mit genau diesen Spalten:

vocabulary_group
term_learning_language
term_user_language
learning_status

Als CSV speichern

  • In Excel: Datei → Speichern unter → CSV (UTF-8)
  • In LibreOffice: Datei → Speichern unter → Dateityp: CSV

Datei hier hochladen

Wähle deine .csv Datei unten aus

Der Converter wandelt sie automatisch in das richtige Format um

JSON herunterladen

Klicke auf „JSON herunterladen"

Die Datei kannst du direkt in der App verwenden

⚠️ Wichtige Hinweise

  • Die erste Zeile muss die Spaltennamen enthalten
  • Die Spaltennamen müssen genau so geschrieben sein
  • Speichere die Datei immer als CSV (UTF-8)
  • Excel kann ; oder , verwenden → das wird automatisch erkannt
