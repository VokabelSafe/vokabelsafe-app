📄 Import-Anleitung für VokabelSafe

Sie können Vokabellisten bequem am PC vorbereiten und anschließend als JSON in VokabelSafe importieren.

Die Umwandlung erfolgt direkt im Browser. Ihre Datei wird dabei nicht an einen Server hochgeladen.

📄 Anleitung: Vokabelliste aus Excel / OpenOffice importieren

Tabelle erstellen

Erstelle in Microsoft Excel oder LibreOffice Calc eine Tabelle mit genau diesen Spalten:

vocabulary_group
term_learning_language
term_user_language
learning_status

Vokabeln eintragen

Beispiel (tabellarisch):

Kurzanleitung:
  1. Tabelle als CSV (UTF-8) speichern.
  2. Die Datei in unserem Online-Converter auswählen.
  3. Erzeugtes JSON herunterladen und in die App importieren.

💡 Tipp: Trenne mehrere Begriffe in einer Zelle einfach mit einem Semikolon (;). Das Trennzeichen der CSV-Datei (Komma oder Semikolon) erkennt der Converter automatisch.
